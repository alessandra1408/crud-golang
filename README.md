# CRUD in Golang(GO)
---

## Stacks to create this project:
- Golang (GO)
- Postgresql

You will need a config.toml file to configure the local variables for this application like this:
``[api]
port = "port_numer"

[database]
host = ""
user = ""
pass = ""
name = "" ``
